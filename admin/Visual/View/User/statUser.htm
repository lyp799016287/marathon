指标解释

日平均PV: 所选日期前14天商品的PV之和/14。

日平均UV: 所选日期前14天商品的UV之和/14。

平均日销量: 所选日期前14天商品的销售数量之和/14。

库存数量：所选时间时的财务数量。

虚库数量：所选时间时的虚拟库存数量。

可用数量：所选时间时的可用数量。

预计销售天数：所选时间时的可用数量/平均日销量。

采购在途数量：商品被生成了采购单且采购单为已审核状态。

累计缺货天数：商品当月累计缺货天数。

当前缺货天数：目前到之前没有缺货的天数。

销售趋势：d1d2d3w1w2w3m1m2m3，所选时间的前一天销售数量/前第二天/前第三天/前一周/前第二周/前第三周/前一月/前第二月/前第三月。

缺货商品：1 商品属性中应为'本仓采购'的商品；2 若该商品在其它仓中支持'本仓销售',在其它仓中也应无货；3 已上架状态且可用库存+虚拟库存小于或等于0。满足以上条件的商品种类数汇总。

热销缺货商品：缺货商品中是热销商品。热销定义：近两周、单仓销售数量排名占本仓前20%且满足DMS在0.05以上的商品and小类最近两周销售数量在该小类TOP10%的商品。

缺货未采购商品：缺货商品中采购在途数量为0的商品。

热销未采购商品：热销商品中，日均在0.5以上，采购在途为0且库存数量减去平均日销量*2的数值小于等于零的商品。

建议采购商品：预计销售天数小于等于7的商品。

热销定义：近两周、单仓销售数量排名占本仓前20%且满足DMS在0.05以上的商品and小类最近两周销售数量在该小类TOP10%的商品。

指标解释

商品sku数（全部）：商品状态为正常、show状态下的商品种类数。

商品sku数（本仓销售）：1 商品状态为正常、show状态下的商品种类数；2 本仓是否采购为是。

缺货sku数：正常且show状态商品中缺货商品的商品种类数。

缺货率：缺货sku数/商品sku数（全部）。

缺货未采购sku数：正常且show状态商品中缺货商品且采购在途数量为0。

热销sku数：正常且show状态商品中热销商品的商品种类数。

热销缺货sku数：热销商品中缺货的商品种类数。

热销缺货率：热销缺货sku数/热销sku数。

热销缺货未采购sku数：正常且show状态商品中热销缺货商品且采购在途数量为0。

实库sku数：正常且show状态商品中，可用数量>=1的商品种类数。
